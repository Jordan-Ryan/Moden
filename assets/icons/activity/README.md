# Activity Icons - Icons8 SF Symbols Style

Download these icons from [Icons8 SF Symbols Regular Style](https://icons8.com/icons/all--os-ios--style-sf-regular) and save them here with the following names:

## Required Icons:

1. **running.svg** - Search for "figure run" or "running" icon
2. **cycling.svg** - Search for "bicycle" or "cycling" icon
3. **hiit.svg** - Search for "bolt" or "lightning" icon
4. **walking.svg** - Search for "figure walk" or "walking" icon
5. **strength.svg** - Search for "dumbbell" or "weight" icon
6. **workout.svg** - Search for "fitness" or "workout" icon

## Download Instructions:

1. Go to https://icons8.com/icons/all--os-ios--style-sf-regular
2. Search for each icon name above
3. Make sure to select the **SF Regular** style (not SF Pro or other styles)
4. Download each icon as **SVG** format
5. Save them in this folder with the exact names listed above

## Icon Recommendations:

- **Running**: `figure.run` or similar running person icon
- **Cycling**: `bicycle` icon
- **HIIT**: `bolt` or `bolt.fill` lightning bolt icon
- **Walking**: `figure.walk` or similar walking person icon
- **Strength**: `dumbbell` or `dumbbell.fill` weight icon
- **Workout**: `figure.fitness` or generic fitness/workout icon

Make sure all icons are white/transparent SVG that will be colored white on the colored backgrounds.
